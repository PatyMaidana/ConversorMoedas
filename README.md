# Desafio - Conversor de Moeda
Esta aplicação é um conversor de moeda, feito na linguagem Java (JDK 21) que consome uma api pública de taxas de câmbio.

API: https://www.exchangerate-api.com/

## Instruções
* O conversor apresenta um menu simples, contendo 7 opções, de 1 a 6 para conversão de moedas (real, dolar, euro e iene) e uma opção para encerrar a aplicação.
* Para utilizar essa aplicação basta baixar os códigos em .zip e abrir com a sua IDE de preferência (eu utilizei o IntelliJ).
* Foi utilizado a biblioteca Gson https://mvnrepository.com/artifact/com.google.code.gson/gson/2.11.0
